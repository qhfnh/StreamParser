About

H.264/H.265 裸码流在线解析器

本项目是一个纯静态前端工具，用于分析 H.264/H.265 Annex B 裸码流中的 NAL 单元、参数集、slice header、SEI、帧类型和二进制字段范围。

适用场景

  • 检查编码器是否输出正确的 SPS/PPS/VPS。
  • 排查 I/P/B/IDR 帧类型和 GOP 结构。
  • 查看协议字段与原始二进制 bit 的对应关系。
  • 验证 H.264/H.265 Annex B 裸码流是否完整。

协议参考

解析逻辑参考 ITU-T H.264 / ISO/IEC 14496-10 与 ITU-T H.265 / ISO/IEC 23008-2。页面展示的 syntax element 尽量保持协议原名，便于与标准文档和其他分析工具交叉验证。

隐私设计

文件在浏览器中通过 Web Worker 解析，不上传到服务器。静态部署时，站点本身不需要后端 API。
